🧠 Gemini AI Agent Chat System

An AI-powered chat application built using Google Gemini API that demonstrates agent-like behavior with tool execution capabilities using Node.js and Express.

🚀 Overview

This project is a simple implementation of an LLM-based AI agent system that can:

Respond to user queries using Gemini AI
Execute tool-based functions (like a calculator tool)
Handle real-time chat interactions via a web interface
Simulate basic agentic AI behavior using structured prompts and tools
⚙️ Features
🤖 AI-powered conversational assistant (Gemini API)
🧠 Tool-based reasoning (calculator tool integration)
💬 Real-time chat UI using EJS
⚡ Backend API built with Node.js + Express
🔄 Async communication between frontend and AI model
🧩 Modular architecture for agent + tools separation
🧠 What Makes This an “AI Agent” Project?

This system goes beyond a simple chatbot by:

Allowing the AI model to use external tools
Structuring responses through agent-like execution flow
Separating reasoning (LLM) from actions (tools)
🛠️ Tech Stack
Node.js
Express.js
Google Gemini API (AI SDK)
EJS (Frontend templating)
JavaScript (ES Modules)
💡 How It Works
User sends a message from UI
Backend receives request via Express
Message is sent to Gemini API
AI generates response (and can call tools)
Response is returned to frontend
UI updates chat in real time
📚 Learning Outcome
LLM API integration
Basic agentic AI design
Tool usage in AI systems
Backend + frontend full-stack flow
Async API handling
